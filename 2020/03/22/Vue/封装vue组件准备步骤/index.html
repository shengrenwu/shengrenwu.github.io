发表于 2020-03-22 更新于 2020-03-28 分类于 vue

创建

  1. 创建自己要封装的组件如Cleandar.vue(进行逻辑编写)
  2. 创建index.js文件夹, 可存放所有封装好的组件
  3. 创建install文件, 将封装好的组件注册到vue全局
  4. 在main.js中导入就可以通过Vue.use(Clendar)进行使用了
